Projects

  1. Superfluidity in neutron matter

    • Solved non-linear integral equations using Khodel's algorithm to compute BCS pairing gap for neutron matter at the neutron star outer core densities.
    • Solved high-dimensional stiff differential equations to study the renormalization flow of the pairing gaps with Similarity renormalization group (SRG).

  2. Hamiltonian renormalization in QFT using Gaussian process regression

    • Designed an algorithm for solving non-linear integro-differential equation using Gaussian process regression (GPR).
    • Optimized stationary covariance matrix computation using tensor decomposition.
    • Used Gaussian process to design stochastic quadrature for computing multi-dimensional intrgrals.
    • Derived uncertainty estimate of the stochastic integration quadrature using central limit theorem for weakly-correlated process.
    • Designed a algorithm for tracking evolution of Gaussian process hyperparameters with the differential equation using numerical homotopy continuation.
    • Implemented Exponential time difference (ETD) method for alleviating stiffness in the integro-differential equation.
    • Constructed a stochastic differential equation prescription to incorporate uncertainty propogation using Itô calculus.
